# init-server-environment
Small Ansible repo that helps me to configure my servers (install needed tools, configurate them, push some secrets)

Files:
 - [init_ansible_dev_env.sh](init_ansible_dev_env.sh) - install basic packages needed to start developing Ansible scripts (wget, ansible, etc.)
- [check_on_localhost.sh](check_on_localhost.sh) - simple bash script to check local run of init_utils.yml
